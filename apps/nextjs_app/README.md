# Next.js (App Router)

Server-rendered shell; client `TodoClient` fetches Express. Includes **`/api/todos`** as a sample API route using `@repo/shared`.

## Environment

- `NEXT_PUBLIC_API_URL` — Express URL for the browser (default `http://localhost:3000`)

## Scripts

- `pnpm dev` — port **3002**
- `pnpm build`
