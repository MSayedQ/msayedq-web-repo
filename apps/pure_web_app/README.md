# Pure HTML / CSS / JavaScript

Vite serves `index.html`, `style.css`, and `script.js`. Styling pulls in `@repo/ui`.

## Environment

- `VITE_API_URL` — Express base URL (default `http://localhost:3000`)

## Scripts

- `pnpm dev` — Vite dev server (port **5181**)
- `pnpm build` — production build to `dist/`

Run the Express server (`servers/express_server`) on port 3000 for API calls.
